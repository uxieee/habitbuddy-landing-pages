## Defects Found and Fixed

### F-001: Accessibility violations on core pages
- Severity: High
- Impact: Screen-reader and keyboard users encountered landmark/semantic issues; axe reported 45 violations on `index.html`, 35 on `maxsupport.html`, 27 on `giftahabitbuddy.html`.
- Fixes implemented:
  - Added `main` landmarks on primary pages.
  - Added `title` to testimonial video iframes.
  - Fixed table header text for comparison table.
  - Fixed modal hidden-state accessibility (`inert` toggling for phone preview modal).
  - Corrected heading structure on gift form section.
  - Adjusted intro animation opacity behavior to avoid inaccessible transient contrast states.
- Files changed:
  - `/Users/uxie/Documents/Vibe Code/Habit Buddy/landing-pages/index.html`
  - `/Users/uxie/Documents/Vibe Code/Habit Buddy/landing-pages/maxsupport.html`
  - `/Users/uxie/Documents/Vibe Code/Habit Buddy/landing-pages/giftahabitbuddy.html`

### F-002: Webhook oversized payload envelope handling order
- Severity: Medium
- Impact: Oversized webhook payloads returned config errors before payload-limit rejection when env vars were missing.
- Fix:
  - Reordered processing in webhook handler to enforce body-size limit before config assertion.
- File changed:
  - `/Users/uxie/Documents/Vibe Code/Habit Buddy/landing-pages/functions/api/stripe-webhook.js`
